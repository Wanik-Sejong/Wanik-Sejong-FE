## 🎤 발표 대본

### 1. 문제 정의 (1분)

> **[슬라이드 1: 문제 상황]**

안녕하세요, 저희는 **완익세종**팀입니다.

여러분, 대학 생활 중 이런 고민 해보신 적 있으신가요?

*"내가 AI 엔지니어가 되고 싶은데, 어떤 과목을 들어야 하지?"*
*"선후수 관계는 어떻게 되지?"*
*"방학에는 뭘 해야 취업에 도움이 될까?"*

저희 팀도 같은 고민을 했고, 주변 친구들도 마찬가지였습니다.

현재 세종대학교 학생들은:
- **수강편람**을 일일이 찾아보며 과목을 선택하고
- **선배들에게 물어보거나** 학과 사무실을 방문하며
- **방학 계획은 막연하게** 세우거나 아예 세우지 못합니다

이런 **비효율적인 진로 설계 과정**을 해결하기 위해 저희는 **완익세종**을 개발했습니다.

### 2. 솔루션 소개 (1분 30초)

> **[슬라이드 2: 서비스 개요]**

완익세종은 **AI 기반 맞춤형 학습 로드맵 추천 서비스**입니다.

#### 핵심 프로세스는 3단계로 매우 간단합니다:

**1단계: 성적표 업로드**
- 세종대학교 포털에서 다운로드한 **기이수성적조회 Excel 파일**을 업로드합니다
- 드래그 앤 드롭만으로 간편하게 업로드 가능합니다

**2단계: 진로 입력**
- 희망하는 진로와 관심 분야를 입력합니다
- 예: "AI/ML 엔지니어", "백엔드 개발자", "데이터 사이언티스트"

**3단계: AI 로드맵 생성**
- Google Gemini AI가 여러분의 이수 과목과 진로를 분석해
- **학기별 추천 과목**과 **방학별 활동 계획**을 자동으로 생성합니다

> **[슬라이드 3: 로드맵 예시 화면]**

생성된 로드맵에는 다음 정보가 포함됩니다:

✅ **현재 역량 분석**: 여러분이 현재 가지고 있는 강점과 부족한 부분
✅ **학기별 추천 과목**: 선후수 관계를 고려한 과목 추천
✅ **방학별 활동 계획**: 온라인 강의, 프로젝트, 자격증, 공모전 추천
✅ **추천 기술스택**: 해당 진로에 필요한 프레임워크, 언어, 도구
✅ **구체적인 조언**: 각 단계별 학습 방법과 팁

### 3. 핵심 기술 (1분 30초)

> **[슬라이드 4: 기술 아키텍처]**

완익세종의 핵심 기술 스택을 소개합니다.

#### Frontend
- **Next.js 16 App Router**: 최신 React 기반 프레임워크
- **TypeScript**: 타입 안전성 보장
- **Tailwind CSS 4**: 세종대학교 브랜드 컬러를 적용한 디자인 시스템

#### Backend & AI
- **Google Gemini 2.0 Flash**: 빠르고 정확한 AI 추론
- **Next.js API Routes**: 서버리스 백엔드 API
- **xlsx 라이브러리**: Excel 파일 파싱

#### 핵심 기능 구현

**1) Excel 파싱 시스템**
```
세종대 성적표 → 10개 필드 자동 추출
(학수번호, 교과목명, 이수구분, 학점, 평점 등)
→ 자동 GPA 계산 → 전공/교양 학점 분류
```

**2) AI 프롬프트 엔지니어링**
- 학생의 이수 과목, 진로 목표, 관심 분야를 종합 분석
- 세종대학교 교과목 정보와 매칭
- 선후수 관계를 고려한 과목 추천
- 실전 역량 강화를 위한 비교과 활동 추천

**3) 이중 모드 아키텍처**
- **개발 모드**: Mock 데이터로 빠른 테스트 가능
- **프로덕션 모드**: 실제 AI API 연동

> **[슬라이드 5: 데이터 흐름도]**

```
Excel 업로드 → 파싱 (10개 필드) → GPA 계산
→ Gemini AI 분석 → 학기별/방학별 로드맵 생성
→ 타임라인 뷰로 시각화
```

## 📊 부록: 예상 질문 & 답변

### Q1. AI가 추천한 과목이 실제로 존재하는 과목인가요?

**A**: 현재는 Gemini AI가 일반적인 컴퓨터공학 교과목을 기반으로 추천하며, 향후 **세종대학교 실제 교과목 데이터베이스**를 연동하여 학수번호까지 정확하게 매칭할 예정입니다.

### Q2. 다른 대학에서도 사용할 수 있나요?

**A**: 네! Excel 파일 포맷만 맞춘다면 **모든 대학교에 적용 가능**합니다. 각 대학의 교과목 정보를 추가로 학습시키면 더 정확한 추천이 가능합니다.

### Q3. 개인정보 보안은 어떻게 처리하나요?

**A**: 업로드된 성적표는 **서버에 저장하지 않고** AI 분석에만 사용한 후 즉시 삭제됩니다. 생성된 로드맵은 **브라우저의 sessionStorage**에만 저장되어 탭을 닫으면 완전히 삭제됩니다.

### Q4. AI 추천이 항상 정확한가요?

**A**: AI는 통계적 패턴을 기반으로 추천하므로 **참고 자료**로 활용하는 것이 좋습니다. 최종 결정은 **학생 본인**과 **지도교수 상담**을 통해 하시길 권장합니다.

### Q5. 향후 개선 계획은?

**A**: 다음 기능들을 추가할 예정입니다:
1. **세종대 교과목 DB 연동** - 실제 학수번호와 과목 정보 매칭
2. **선배 멘토링 매칭** - 같은 진로의 선배와 연결
3. **로드맵 공유 기능** - 친구들과 로드맵 공유 및 비교
4. **진행률 추적** - 로드맵 실천 현황 체크리스트
5. **채용 공고 연동** - 기업 JD 분석 후 필요 역량 추가 학습
