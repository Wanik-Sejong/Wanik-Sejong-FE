## 📌 예상 질문 & 답변

### Q1: "AI가 추천한 로드맵이 얼마나 정확한가요?"

**A:**
"Google Gemini 2.0을 사용하고 있고, 프롬프트 엔지니어링을 통해 세종대 교과과정과 실무 수요를 반영하도록 학습시켰습니다. 현재 정확도는 약 85-90% 수준이며, 사용자 피드백을 통해 지속적으로 개선하고 있습니다."

---

### Q2: "다른 대학에서도 사용할 수 있나요?"

**A:**
"네, 가능합니다. 엑셀 파일 포맷만 맞추면 다른 대학에서도 사용할 수 있습니다. 다만 현재는 세종대 교과과정에 최적화되어 있어서, 다른 대학은 프롬프트 수정이 필요합니다."

---

### Q3: "개인정보 보호는 어떻게 하나요?"

**A:**
"성적표 데이터는 서버에 저장하지 않고, 브라우저의 세션 스토리지에만 임시 저장됩니다. 페이지를 닫으면 자동으로 삭제되고, AI 분석을 위해 전송된 데이터도 API 응답 후 즉시 파기됩니다."

---

### Q4: "비용은 얼마인가요?"

**A:**
"현재는 무료 베타 서비스입니다. Google Gemini API 사용료가 발생하긴 하지만, 학생 1명당 로드맵 1회 생성 비용이 약 10-20원 정도로 매우 저렴합니다."

---

### Q5: "실시간으로 성적이 업데이트되면 로드맵도 자동 업데이트되나요?"

**A:**
"현재는 수동 업로드 방식이지만, 향후 유포털 API 연동을 통해 실시간 동기화를 계획하고 있습니다. 성적이 바뀔 때마다 알림을 보내고, 로드맵을 자동으로 업데이트하는 기능을 개발 중입니다."
